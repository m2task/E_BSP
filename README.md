# E_BSP

これはTrading Card GameのBattle Spiritsをシミュレーションできるプロジェクトです。
https://m2task.github.io/E_BSP/
## 特徴
- デッキをローカルストレージに保存
- バトル画面でカードやコアをドラッグ＆ドロップ
- GitHub Pagesで動作する

## 使い方
1. index.html でデッキを構築
2. 「対戦へ」ボタンで battle.html に遷移
3. デッキが自動で反映されてバトル開始！
